# configs 协作规范

## 1. 目录职责

- 本目录存放阶段配置文件, 默认命名为 `phaseN.yaml`.
- 配置键名使用英文, 单位写进键名或注释, 例如 `pocket_cutoff_angstrom`.
- 不写入密钥, 绝对本地路径, 机器名或用户私有路径.

## 2. 修改规则

- 新增配置时同步检查对应 `scripts/phaseN_*.py` 和 `src/` 中读取配置的代码.
- 默认值应可复现, 随机种子优先显式写入配置.
- 配置只描述参数, 不承载长篇方案说明; 方案说明放入 `docs/`.
- 阶段 0 配置保持最小可运行, 并与阶段入口和源码读取逻辑一致.
